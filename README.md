# Logan Services — Lead Generation Demo

Redesign demo for [Logan Services](https://www.logan-inc.com) — HVAC, plumbing & indoor air quality company. Family-owned since 1969, serving Dayton, Columbus, Cincinnati & Northern Kentucky.

**Live Demo:** https://lead-demo-logan-inc-com.pages.dev
**Original Site:** https://www.logan-inc.com

## Tech

Static HTML + CSS + vanilla JS. Single shared stylesheet (`assets/css/shared.css`) and script (`assets/js/shared.js`). Google Fonts (Inter) for typography. No build step, no framework.

## Pages

- `index.html` — Home (hero, services overview, testimonials, CTA)
- `about.html` — Company story, team, values
- `service.html` — Service detail pages (HVAC, plumbing, air quality)
- `contact.html` — Contact form + location info

## Deploy

Cloudflare Pages — push to `main` triggers auto-deploy.
